Запустил Wave 2 на проде — my.azarean.ru. Расскажу что нового.

Раньше был просто слайдер боли 0-10. Теперь полноценный дневник:
   • Локализация — патиент тапает на конкретное место (передняя

   • Интенсивность 0-10 + словесная характеристика (Терпимо/Сильная

   • Когда болит — в покое, при сгибании, ночью, при ходьбе
   • Характер — ноющая, острая, жгучая, простреливающая, пульсирующая

   Плюс отдельная кнопка "Записать резкую боль" на главной — это
   быстрая запись (Pain Event), для редких острых эпизодов между
   обычными записями.

   КРАСНЫЕ ФЛАГИ. Если патиент отмечает локализацию из списка
   опасных (для колена — икроножная мышца = подозрение на ТГВ; для
   плеча — задняя поверхность шеи = подозрение на цервикальную
   радикулопатию), нам приходит срочный push в Telegram. Так мы
   ловим экстренные ситуации сразу, не ждём планового визита.

📐 Замеры (новая вкладка для патиента)
   Патиент может сам записывать свои параметры между визитами:
   • ROM в градусах (сгибание/разгибание колена, abduction/flexion
     плеча и т.д.) — для одной или обеих сторон сразу
   • HBB ("рука за спину") — категориально, до какого позвонка
     достаёт большой палец. От T1 (отлично) до сакрум/большой
     вертел (плохо). Это стандартный клинический тест для плеча.
   • Окружности (для отёка/гипотрофии) в см — например над/под
     надколенником, посередине бицепса

     отёка / положения. Перед первой загрузкой просим согласие

   нельзя (это Tier 3, отложили в Wave 3). Но base flow готов.

⚖️  Критерии перехода между фазами
   На моей стороне (в админке) можно настроить для каждой фазы:
   - какие замеры должны выйти на определённый уровень (например

   - какие чек-листы патиент сам должен пройти ("я могу пройти

   Для ACL-программы я уже заложил ~29 default критериев по AAOS

🔧 Под капотом всякое скучное

## Подсказки про tone

- Не использовать слово «**пациент**» в текстах для пилота-клиентов (только для нас внутри). По 152-ФЗ + коммерческое позиционирование «**клиент**» / «**патиент**» (последний — наша внутренняя нейтральность).
- Не использовать «**лечение**» / «**терапия**» — это медицинская услуга = должна быть лицензия. Используй «**программа восстановления**» / «**тренировки**» / «**сопровождение**».
- Можно «**прогресс**» / «**восстановление**» / «**мобильность**» — нейтрально.
- Disclaimer-овой клаузы «не является медицинской услугой» **в SMS/IM не нужна** — она уже в самом приложении при первом логине.
