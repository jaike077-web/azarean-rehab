## patient-dashboard-v2.jsx

Single-file прототип пациентского дашборда (783 строки, 18 компонентов).
Создан 2026-02-10, до того как компоненты были разнесены по отдельным файлам
в `frontend/src/pages/PatientDashboard/components/`.

Содержит изначальную задумку: цвета, анимации, порядок секций, тексты FAQ,
структуру фаз реабилитации (ACL), тесты для PainSlider и DiaryHistory.
