# Диагностическая форма — коленный сустав

Одностраничная форма для первичного диагностического обследования пациента. Предназначена для работы на планшете и десктопе, поддерживает автосохранение, валидацию и генерацию PDF-отчёта.

## Быстрый старт

1. Откройте `index.html` в браузере.
2. Заполните форму.
3. Нажмите «Сгенерировать отчёт (PDF)» для выгрузки файла.

## Структура

```
/diagnostic-form-knee/
├── index.html
├── css/
│   └── styles.css
├── js/
│   ├── main.js
│   ├── validation.js
│   ├── storage.js
│   └── pdf-generator.js
└── assets/
```

## Автосохранение

- Срабатывает каждые 30 секунд.
- Ключ localStorage: `azarean_diagnostic_form_knee`.
- При повторном открытии страницы предлагается восстановить данные.

## Встраивание в Tilda

**Embed-код (iframe):**

```html
<iframe
  src="/path-to-form/diagnostic-form-knee/index.html"
  width="100%"
  height="1200"
  style="border: none;"
  title="Диагностическая форма"
></iframe>
```

**Важно:** для корректной генерации PDF и автосохранения форма должна быть доступна по HTTPS.

## Примечания

- PDF формируется с помощью `jsPDF` и `html2canvas`.
- Для кириллицы при необходимости можно подключить кастомный шрифт в `pdf-generator.js`.
